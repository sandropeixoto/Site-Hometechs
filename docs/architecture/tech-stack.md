# Architecture: Tech Stack

## Core Technologies
- **Framework**: React 18
- **Build Tool**: Vite 5
- **Styling**: Tailwind CSS 4.0 (CSS-first configuration)
- **Language**: TypeScript 5
- **Animations**: Framer Motion

## Design Tokens (Draft)
- **Primary Color**: `#0056b3` (Blue - Security/Trust)
- **Secondary Color**: `#f8f9fa` (White/Light Gray)
- **Accent Color**: `#ffc107` (Warning/Attention)
- **Typography**: Inter / Outfit
- **Effects**: Backdrop blur (glassmorphism), subtle gradients.

## Development Standards
- Functional components with Hooks.
- Container/Presenter pattern when applicable.
- Semantic HTML5 for SEO.
