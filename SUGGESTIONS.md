# SUGGESTIONS.md - Hometechs Project

Sugestões para evoluções futuras do projeto, focando em escalabilidade, SEO e experiência do usuário.

## 🚀 Melhorias Técnicas
1. **Next.js Migration**: Caso o site cresça com um blog ou muitas páginas dinâmicas, migrar para Next.js para melhor SEO (SSR/ISR).
2. **PWA**: Adicionar um manifest e service worker para que o site possa ser "instalado" como um app no celular dos clientes.
3. **i18n**: Preparar para múltiplos idiomas caso a Hometechs expanda para além do mercado local.

## 🎨 Design & UX
1. **Lottie Animations**: Substituir alguns ícones estáticos por animações Lottie no portfólio de soluções.
2. **Interactive Map**: Integrar a API do Google Maps no rodapé com estilo "dark" personalizado.
3. **WhatsApp Floating Button**: Adicionar um botão fixo de WhatsApp com uma animação de "batida de coração" para aumentar a conversão.

## 📈 SEO & Performance
1. **Image Optimization**: Usar formatos modernos como WebP/Avif para todas as imagens (já iniciado com os assets básicos).
2. **Structured Data**: Adicionar `JSON-LD` para que o Google entenda melhor o negócio local (Hometechs) e seus serviços.
3. **Analytics**: Integrar Google Analytics 4 ou Plausible para monitorar o comportamento dos usuários.
